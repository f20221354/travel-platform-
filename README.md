    <h1>Travel Beyond Boundaries <span class="badge">V 1.0</span></h1>
    <p>A high-conversion, dark-themed landing page designed for a modern travel agency. This project focuses on visual storytelling through high-quality imagery, interactive carousels, and strong social proof.</p>

    <h2>🚀 Key Features</h2>
    <ul>
        <li><strong>Hero Experience:</strong> Dynamic background with a "Watch our Video" integration and a multi-parameter search/booking bar (Destination, Dates, Members).</li>
        <li><strong>International Showcases:</strong> A dedicated section for global travel with image-focused cards for Turkey, Bali, and Thailand, including star ratings.</li>
        <li><strong>Regional Spotlight:</strong> An exploration gallery specifically for South India (Coorg, Mysore, Hampi, etc.) with a grid-based layout.</li>
        <li><strong>Social Proof:</strong> Integrated "Google Reviews" slider to build trust with potential customers.</li>
        <li><strong>Data Analytics:</strong> A high-impact stats bar showcasing 200K+ satisfied customers and 180+ tour captains.</li>
    </ul>

    <h2>🎨 Design Specifications</h2>
    <ul>
        <li><strong>Primary Theme:</strong> Deep Black (<code>#000000</code>).</li>
        <li><strong>Accent Colors:</strong> Cyan (Header/Buttons) and Golden Yellow (Highlights/CTA).</li>
        <li><strong>Typography:</strong> Clean, sans-serif fonts for readability against dark backgrounds.</li>
        <li><strong>Layout:</strong> Responsive grid system with emphasis on mobile-first navigation.</li>
    </ul>

    <h2>📂 Project Structure</h2>
    <ul>
        <li><code>Header:</code> Sticky navigation with quick links to International and South India sections.</li>
        <li><code>International Section:</code> 3-column carousel with hover effects and destination summaries.</li>
        <li><code>Reviews:</strong> Horizontal scroll section featuring customer testimonials.</li>
        <li><code>South India Grid:</code> Mixed-size image gallery featuring regional landmarks.</li>
        <li><code>Footer:</code> Detailed company metrics and essential contact information.</li>
    </ul>

    <h2>🛠️ Tech Stack (Suggested)</h2>
    <ul>
        <li><strong>Frontend:</strong> HTML5, CSS3 (Tailwind CSS for utility-first styling).</li>
        <li><strong>Interactivity:</strong> JavaScript (Swiper.js for carousels, Framer Motion for scroll animations).</li>
        <li><strong>Icons:</strong> FontAwesome or Lucide React.</li>
    </ul>
