# Travel Beyond Boundaries `V 1.0`

A high-conversion, dark-themed landing page designed for a modern travel agency. This project focuses on visual storytelling through high-quality imagery, interactive carousels, and strong social proof.

---

## 🚀 Key Features

* **Hero Experience:** Dynamic background with a "Watch our Video" integration and a multi-parameter search/booking bar (Destination, Dates, Members).
* **International Showcases:** A dedicated section for global travel with image-focused cards for Turkey, Bali, and Thailand, including star ratings.
* **Regional Spotlight:** An exploration gallery specifically for South India (Coorg, Mysore, Hampi, etc.) with a grid-based layout.
* **Social Proof:** Integrated "Google Reviews" slider to build trust with potential customers.
* **Data Analytics:** A high-impact stats bar showcasing 200K+ satisfied customers and 180+ tour captains.

## 🎨 Design Specifications

* **Primary Theme:** Deep Black (`#000000`)
* **Accent Colors:** Cyan (Header/Buttons) and Golden Yellow (Highlights/CTA)
* **Typography:** Clean, sans-serif fonts for readability against dark backgrounds
* **Layout:** Responsive grid system with emphasis on mobile-first navigation

## 🛠️ Tech Stack
* **Frontend:** HTML5, CSS3 (Tailwind CSS suggested)
* **Interactivity:** JavaScript (Swiper.js for carousels)
